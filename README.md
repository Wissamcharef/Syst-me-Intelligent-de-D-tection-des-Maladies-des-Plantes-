# Syst-me-Intelligent-de-D-tection-des-Maladies-des-Plantes-
Ce projet consiste à développer une application web basée sur une architecture de microservices intégrant un modèle de Deep Learning pour détecter automatiquement les maladies des plantes à partir d’images, en fournissant un diagnostic rapide et fiable à l’utilisateur.
